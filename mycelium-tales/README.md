# Mycelium Tales

Projeto base do MVP em Godot 4 para o vertical slice de `Manto de Veuspora`.

## Estrutura atual

- `scenes/` contem mundo, entidades, inimigos e UI placeholder
- `scripts/` contem movimento, dialogo, combate e integracao com Ollama/SQLite
- `data/` contem NPCs, biomas, fallbacks e inimigos
- `assets/` esta pronta para receber sprites, fontes e audio
